
# Stockman

Welcome to Stockman! Stockman is a command line tool to help you calculate your return rate and drawdown rate for Stocks.

## Solution - How to run the program ?
Go to the root folder:
  ```ruby
  cd stockman
  bundle install
  ```

Start the stockman:
  ```ruby
  ruby stockman_start.rb
  ```

## How To Test the App ?
  ```ruby
  rspec spec
  ```
